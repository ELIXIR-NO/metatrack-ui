# MetaTrack Webapp — Claude Code Guide

## Project Overview

MetaTrack GUI is the web frontend for the MetaTrack metadata management platform, built for life scientists in Norway (ELIXIR Norway). It lets users manage scientific projects, samples, and assays with rich metadata tracking.

- **Production URL**: metatrack.no
- **API**: https://api.metatrack.no/api
- **Auth server**: https://auth.metatrack.no (Keycloak)

## Tech Stack

- **Framework**: React 19 + TypeScript, built with Vite
- **Routing**: TanStack Router (file-based, auto code splitting)
- **Server state**: TanStack React Query
- **UI**: shadcn/ui (new-york style) + Radix UI + Tailwind CSS v4
- **Forms**: React Hook Form + Zod v4
- **Tables**: TanStack React Table v8
- **Charts**: Recharts v3 + D3
- **Auth**: Keycloak 26 (OIDC + PKCE)
- **Icons**: Lucide React + Tabler Icons
- **Notifications**: Sonner
- **Animation**: Motion

## Key Conventions

### Code Style
- **Tabs** for indentation (tab width 2), enforced by Prettier
- **Double quotes** for strings, semicolons required
- Trailing commas: ES5 style
- Tailwind classes sorted by Prettier plugin

### TypeScript
- Strict mode enabled
- No unused locals/parameters
- Path alias: `@/*` → `./src/*`
- Target: ES2020

### Components
- Function components with hooks only
- Props interfaces defined above the component
- PascalCase for component names, camelCase for functions/hooks

### API Calls
- Use `api-keycloak.ts` for authenticated endpoints (injects Bearer token)
- Use `api-client.ts` for public/basic-auth endpoints
- Error handling via try-catch; use Sonner toasts for user feedback
- Batch operations use `Promise.allSettled`

### Forms
- Always use React Hook Form + Zod schema validation
- Use `@hookform/resolvers/zod` for integration
- Form fields wrapped with the `form-field.tsx` helper component

### State Management
- Server state: TanStack React Query (no Redux/Zustand)
- Auth/theme state: React Context
- UI state: local `useState`

### Routing
- Routes are file-based under `src/routes/`
- `routeTree.gen.ts` is auto-generated by TanStack Router — never edit manually
- Protected routes check auth via context + `useNavigate`

### Styling
- Tailwind CSS v4 utility classes only; no CSS-in-JS
- CSS variables for theme colors (dark/light mode via `next-themes`)
- Component variants via `class-variance-authority`

## Authentication Flow

1. Keycloak initializes on app load (check-SSO)
2. PKCE (S256) flow for token acquisition
3. Token auto-refreshed every 10s (30s threshold)
4. Bearer token injected via `api-keycloak.ts`

## Dev Proxy (Vite)

In development, Vite proxies:
- `/keycloak` → `https://auth.metatrack.no`
- `/api` → `https://api.metatrack.no`

## shadcn/ui Components

- Style: `new-york`, base color: `neutral`, CSS variables enabled
- Icon library: `lucide`
- Add new components via `bunx shadcn@latest add <component>`
- Files land in `src/components/ui/` — treat as primitives, extend via composition

## No Test Framework

There is currently no test framework configured in this project.
